OPCIONES DE INSTALACION Y VENTA - TaskManager
==============================================


OPCIONES DE "INSTALACION"
--------------------------

OPCION 1 — Ejecutable de escritorio (mas profesional)
------------------------------------------------------
Conviertes la app en una app de escritorio real con Tauri (ligero, ~5MB)
o Electron (mas conocido, ~100MB). El cliente la instala como cualquier
programa (.exe, .dmg, .deb).

Comandos:
  npm install -D @tauri-apps/cli @tauri-apps/api
  npx tauri init
  npx tauri build  --> genera instalador .exe / .deb / .dmg

El cliente recibe un instalador, hace doble clic, y listo. Los datos
siguen en localStorage (o puedes migrar a SQLite local con Tauri).


OPCION 2 — App web en hosting (sin instalar nada)
--------------------------------------------------
El cliente accede desde el navegador. Subes el build a:

  Servicio   | Costo          | Facilidad
  -----------|----------------|---------------------
  Vercel     | Gratis (hobby) | vercel deploy
  Netlify    | Gratis         | Drag & drop de /dist
  VPS propio | ~$5/mes        | nginx sirviendo /dist

Comandos:
  npm run build   --> genera carpeta /dist
  luego subes /dist a donde quieras


OPCION 3 — Archivo local (mas simple para un solo cliente)
-----------------------------------------------------------
Ejecutas:
  npm run build

Le entregas la carpeta /dist. El cliente abre index.html con doble clic
en el navegador. Funciona sin servidor porque es todo static. Los datos
quedan en su navegador.

Limitacion: los datos no se comparten entre PCs ni navegadores.
